                                       Knowledge-driven Dialogue
----------------------------------------------------------------------------------------------------------------
This is a pytorch implementation of generative-based model for knowledge-driven dialogue


Requirements
-----------------------------------------------
cuda=9.0
cudnn=7.0
python>=3.6
pytorch>=1.0
tqdm
numpy
nltk
scikit-learn


Quickstart
-----------------------------------------------
Step 1: Train:
train model with the following commands.
	sh run_train.sh

Step 2: Test
test model with the following commands.
	sh run_test.sh


Note: the script run_train.sh/run_test.sh shows all the processes including data processing and model training/testing.
      be sure to read it carefully and follow it.
